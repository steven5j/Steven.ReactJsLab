## 如何啟用這個範例

1.用`VS CODE` 打開到範例資料夾

2.在終端機(cmd)下 `npm install`指令，開始在資料夾下安裝npm

2.在終端機(cmd)下 `npm run server` 指令，會啟動一個簡易的資料庫後端JSON

3. 在終端機(cmd)下 `npm run start` 指令，會詢問是否要創立不同的port去啟動，選擇Y

4.啟用範本
